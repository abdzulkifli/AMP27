Dashboard Methodology

How to Read the AMP2027 Dashboard

Effective Cost

Uses the numeric post-challenge estimate. Where that field is TBC, the original estimate is used provisionally and marked with an asterisk.

Strategic Readiness

A derived completeness score using seven register checks. It is a data-quality indicator, not an approval or delivery-performance rating.

Management Attention

Clickable lenses identify status, cost certainty, classification, timeline, and material cost-variance issues already present in the source data.

Cost Variance

Compares the effective dashboard cost with the original estimate. The attention lens flags absolute movement above 10% where the original estimate is greater than zero.

Quarterly Delivery Load

Uses the earliest and latest recognised DD/MM/YYYY dates in each action plan. It does not infer progress or lateness.

Cost–Benefit Matrix

Shows source CBA ratios against effective cost. ICT classification appears in the tooltip; bubble size is scaled for readability.

Budget Journey

Proposed and approved budget values are displayed only when populated in the source. Missing budget fields are not estimated or fabricated.

Governance Data Gaps

Owner, decision owner, progress percentage, RAG status, and formal decision-required fields are not currently in the source and are therefore not invented.
